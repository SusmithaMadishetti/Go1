Steps to write a GO program:
1.Install Go from https://golang.org/
2.Set environment variables
3.Use any text editor.
4.get Go extensions in text editor
5.create a file using go extension .go
6.save it and run using --> go run filename.go

Other commands are:
1.go build filename.go
2..\filename.exe

to check version --> go version
to find a bug report --> go bug
